# StockPrice
#### How to run with Virtualenv:
1. git clone
2. cd StockPrice
3. virtualenv venv
4. source venv/bin/activate
5. pip install -r requirements.txt
6. flask run
#### How to run using Docker:
1. git clone
2. cd StockPrice
3. docker compose up
